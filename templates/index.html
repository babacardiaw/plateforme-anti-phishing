Qu'est-ce que le phishing?

Le phishing est une technique utilisée par les cybercriminels pour voler vos informations personnelles en se faisant passer pour des entités légitimes.

Comment reconnaître un phishing?

  • URL suspecte ou mal orthographiée
  • Demande d'informations personnelles
  • Sentiment d'urgence dans le message
  • Erreurs grammaticales ou orthographiques
